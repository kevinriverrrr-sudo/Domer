   • Карточки команд с формами ввода

   • Кнопки для быстрого доступа к командам

3. СПРАВОЧНИК
   • Полный список всех команд сервера
   • ID оружий
   • Полезные советы
   • Удобный поиск по тексту

4. ИСТОРИЯ КОМАНД
   • Сохранение всех сгенерированных команд
   • Двойной клик для копирования
   • Очистка истории

🚀 КАК СКОМПИЛИРОВАТЬ В EXE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Откройте build_windows.bat на Windows
2. Дождитесь завершения компиляции
3. EXE файл будет в папке dist\

Команда компиляции:
pyinstaller --onefile --windowed --name=SAMP_Arizona_RP_Admin_Tools --noconsole admin_tools.py
