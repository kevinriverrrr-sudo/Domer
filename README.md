# Telegram бот с интеграцией Gemini AI

Telegram бот, использующий нейросеть Gemini AI для ответов на сообщения пользователей.

## Возможности

- Ответы на текстовые сообщения с помощью Gemini AI
- Команды `/start` и `/help`
- Автоматическая обработка длинных ответов (разбиение на части)

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите бота:
```bash
python3 telegram_bot.py
```

## Конфигурация

Токены API настроены в файле `telegram_bot.py`:
- `TELEGRAM_BOT_TOKEN` - токен Telegram бота
- `GEMINI_API_KEY` - API ключ Gemini AI

## Использование

После запуска бота:
1. Найдите вашего бота в Telegram
2. Отправьте команду `/start`
3. Начните общаться с ботом - он будет отвечать с помощью Gemini AI

## Статус

Бот запущен и работает в фоновом режиме.
