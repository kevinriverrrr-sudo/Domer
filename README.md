# SAMP Arizona RP Admin Tools

Программа для администрирования сервера SAMP Arizona RP.

## Возможности

- ✅ Подключение к серверу через RCON
- ✅ Управление игроками (kick, ban, teleport)
- ✅ Выдача денег и оружия
- ✅ Выполнение RCON команд
- ✅ Просмотр списка игроков
- ✅ Управление сервером
- ✅ Система логов

## Установка

1. Установите Python 3.8 или выше
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование

### Запуск из исходников:
```bash
python admin_tools.py
```

### Компиляция в EXE:

**Windows:**
```bash
build.bat
```

**Linux/Mac:**
```bash
chmod +x build.sh
./build.sh
```

После компиляции EXE файл будет находиться в папке `dist/SAMP_Arizona_RP_Admin_Tools.exe`

## Настройка

1. Запустите программу
2. Введите IP сервера, порт и RCON пароль
3. Нажмите "Подключиться"
4. Настройки сохраняются автоматически в файл `config.json`

## Требования

- Python 3.8+
- Доступ к серверу SAMP с RCON
- RCON пароль от сервера

## Команды

Программа поддерживает все стандартные RCON команды SAMP, а также включает быстрые команды для:
- Управления игроками
- Выдачи предметов
- Управления сервером

## Лицензия

Свободное использование для SAMP Arizona RP серверов.
