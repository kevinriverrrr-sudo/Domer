# SAMP Arizona RP Admin Tools Helper

Общедоступный помощник для админов сервера SAMP Arizona RP.

## Возможности

- ✅ Генератор команд с формами ввода
- ✅ Быстрые команды для частых действий
- ✅ Полный справочник команд сервера
- ✅ История команд для повторного использования
- ✅ Автоматическое копирование в буфер обмена
- ✅ Работает БЕЗ подключения к серверу
- ✅ Не требует RCON пароля

## Установка

1. Установите Python 3.8 или выше
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование

### Запуск из исходников:
```bash
python admin_tools.py
```

### Компиляция в EXE:

**Windows:**
```bash
build_windows.bat
```

**Linux/Mac:**
```bash
chmod +x build.sh
./build.sh
```

После компиляции EXE файл будет находиться в папке `dist/SAMP_Arizona_RP_Admin_Tools.exe`

## Как использовать

1. Запустите программу
2. Выберите нужную команду из вкладок
3. Заполните параметры (если требуется)
4. Команда будет скопирована в буфер обмена
5. Вставьте команду в консоль сервера (Ctrl+V)

## Требования

- Python 3.8+ (для запуска из исходников)
- Windows (для компиляции EXE)

## Особенности

- **Не требует подключения к серверу** - программа работает полностью офлайн
- **Не требует RCON пароля** - просто генерирует команды для копирования
- **Общедоступный помощник** - может использоваться любым админом
- **Удобный интерфейс** - всё в одном месте

## Лицензия

Свободное использование для SAMP Arizona RP серверов.
