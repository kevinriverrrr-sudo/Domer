# Telegram Virus Scanner Bot

Telegram бот для проверки файлов и ссылок на вирусы через VirusTotal API.

## Возможности

- ✅ Проверка файлов через все антивирусы VirusTotal (более 70 антивирусов)
- ✅ Проверка ссылок (URL) через все антивирусы
- ✅ Детальные результаты от каждого антивируса
- ✅ Красивое форматирование результатов
- ✅ Автоматическое определение статуса безопасности

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python3 virus_scanner_bot.py
```

## Использование

1. Найдите бота в Telegram
2. Отправьте команду `/start` для начала работы
3. Отправьте файл или ссылку для проверки
4. Дождитесь результатов анализа

## Команды

- `/start` - Начать работу с ботом
- `/help` - Справка по использованию

## Технические детали

- Максимальный размер файла: 32 МБ (лимит VirusTotal)
- Использует VirusTotal API v3
- Асинхронная обработка запросов
- Автоматическое ожидание завершения анализа
