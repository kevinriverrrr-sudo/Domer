# Telegram Bot для cto.new

Telegram бот для работы с кодами cto.new.

## Функционал

### Для пользователей:
- ?? **Получить код** - получение кода с подтверждением
- ?? **Поделиться кодом** - возможность поделиться кодом с другими пользователями
- ?? **Смена языка** - поддержка русского и английского языков (команды `/language` или `/lang`)

### Для администратора:
- ?? Просмотр всех поделенных кодов
- ?? Блокировка пользователей
- ? Разблокировка пользователей
- ?? Включение/выключение режима технического обслуживания
- ? Включение/выключение бота

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите бота:
```bash
python3 bot.py
```

## Команды

- `/start` - начать работу с ботом
- `/admin` - открыть админ панель (только для администратора)
- `/language` или `/lang` - изменить язык интерфейса
- `/cancel` - отменить текущее действие

## Админ панель

Доступна по команде `/admin` только для администратора (ID: 7694543415)

## Файлы данных

- `shared_codes.json` - хранит поделенные коды
- `blocked_users.json` - хранит список заблокированных пользователей
- `bot_status.json` - хранит статус бота (включен/выключен, режим обслуживания)
- `user_languages.json` - хранит выбранные языки пользователей
