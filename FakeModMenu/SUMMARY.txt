🚀 СЛЕДУЮЩИЕ ШАГИ
═══════════════════════════════════════════════════════════════════

ШАГ 1: Создайте ключ для подписи APK
───────────────────────────────────────────────────────────────────
$ cd /workspace/FakeModMenu
$ ./create_keystore.sh

При создании введите пароль (например: ModMenu2025!)
Запомните этот пароль - он понадобится для обновлений!

ШАГ 2: Создайте файл keystore.properties
───────────────────────────────────────────────────────────────────
$ nano keystore.properties

Впишите в него:
storePassword=ваш_пароль_из_шага_1
keyPassword=ваш_пароль_из_шага_1
keyAlias=modmenu
storeFile=keystore.jks

ШАГ 3: Соберите подписанный APK
───────────────────────────────────────────────────────────────────
$ ./gradlew assembleRelease

Готовый APK будет здесь:
app/build/outputs/apk/release/app-release.apk

ШАГ 4: Установите на Android устройство
───────────────────────────────────────────────────────────────────
Через USB:
$ adb install app/build/outputs/apk/release/app-release.apk

Или:
• Скопируйте APK на телефон
• Откройте файл на телефоне
• Разрешите установку из неизвестных источников

📱 КАК ПОЛЬЗОВАТЬСЯ ПРИЛОЖЕНИЕМ
═══════════════════════════════════════════════════════════════════

5. Перетаскивайте окно за верхнюю часть
6. Переключайтесь между вкладками
7. Включайте/выключайте функции переключателями
8. Настраивайте параметры ползунками
9. Кнопка "×" скрывает/показывает меню
10. Кнопка "Остановить" закрывает меню полностью
