📱 ОПИСАНИЕ ПРИЛОЖЕНИЯ
══════════════════════════════════════════════════════════════════════
Это полнофункциональное Android приложение, которое демонстрирует
визуальный интерфейс "мод меню" с возможностью отображения поверх
других приложений. Все функции "читов" являются неработающими и
представлены только в виде UI элементов.

⚠️  ВАЖНО: Это НЕ настоящий чит для Standoff 2 или любой другой игры!
Приложение создано исключительно в образовательных целях.

✨ ОСНОВНЫЕ ВОЗМОЖНОСТИ
══════════════════════════════════════════════════════════════════════
✓ Главный экран с кнопками запуска/остановки
✓ Плавающее окно меню (оверлей) поверх других приложений
✓ Перетаскиваемое окно меню
✓ Кнопка скрытия/показа меню
✓ 5 функциональных вкладок с настройками
✓ Переключатели (Switch) для включения функций
✓ Слайдеры (SeekBar) для настройки параметров
✓ Современный темный UI дизайн

📦 ПРОЦЕСС СБОРКИ
══════════════════════════════════════════════════════════════════════
1. Создание keystore:      ./create_keystore.sh
2. Настройка конфига:      keystore.properties
3. Сборка Release APK:     ./gradlew assembleRelease
4. Установка:              adb install app-release.apk

Результат: Подписанный APK готовый к распространению

🚀 БЫСТРЫЙ СТАРТ
══════════════════════════════════════════════════════════════════════
$ cd /workspace/FakeModMenu
$ ./create_keystore.sh
$ nano keystore.properties    # Введите пароли
$ ./gradlew assembleRelease
$ adb install app/build/outputs/apk/release/app-release.apk
