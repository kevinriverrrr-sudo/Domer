│ 🚀 БЫСТРАЯ УСТАНОВКА                                           │
└────────────────────────────────────────────────────────────────┘

1️⃣  Откройте браузер Chrome/Edge/Brave

2️⃣  Введите в адресной строке:
    Chrome:  chrome://extensions/
    Edge:    edge://extensions/
    Brave:   brave://extensions/

3️⃣  Включите "Режим разработчика" (переключатель справа вверху)

4️⃣  Нажмите "Загрузить распакованное расширение"

5️⃣  Выберите папку SViris

6️⃣  Готово! Найдите иконку SViris на панели инструментов

📁 Проверить файл:
   → Откройте SViris
   → Вкладка "Файлы"
   → Перетащите файл или нажмите "Выберите файл"

🔗 Проверить ссылку:
   → Вкладка "Ссылки"
   → Введите URL
   → Нажмите "Проверить"

   ИЛИ щелкните правой кнопкой на ссылке
   → "Проверить ссылку с SViris"

👁️ Мониторинг:
   → Вкладка "Поведение"
   → Настройте параметры отслеживания
   → Наблюдайте за активностью в реальном времени

┌────────────────────────────────────────────────────────────────┐
│ 🔧 НАСТРОЙКА API (опционально)                                 │
└────────────────────────────────────────────────────────────────┘

Расширение уже настроено с рабочим API ключом VirusTotal.

Если хотите использовать свой ключ:
1. Зарегистрируйтесь на virustotal.com
2. Получите API ключ
3. Откройте файл config.js
4. Замените значение VIRUSTOTAL_API_KEY
5. Обновите расширение (chrome://extensions/ → ⟳)

❌ Не загружается:
