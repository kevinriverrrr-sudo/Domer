🔹 ШАГ 1: Откройте браузер
   Chrome:  наберите  chrome://extensions/
   Edge:    наберите  edge://extensions/
   Brave:   наберите  brave://extensions/

🔹 ШАГ 2: Включите "Режим разработчика"
   Переключатель в правом верхнем углу

🔹 ШАГ 3: Загрузите расширение
   Нажмите "Загрузить распакованное расширение"
   Выберите папку: SViris

🔹 ШАГ 4: Готово!
   Иконка SViris появится на панели инструментов
   Если не видно - нажмите на иконку пазла и закрепите

Расширение уже настроено с рабочим VirusTotal API ключом!

Если хотите использовать свой ключ:

1. Зарегистрируйтесь на https://www.virustotal.com/
2. Получите бесплатный API ключ
3. Откройте файл config.js
4. Замените значение VIRUSTOTAL_API_KEY на ваш ключ
5. Перезагрузите расширение в chrome://extensions/

Лимиты бесплатного API:
• 4 запроса в минуту
• 500 запросов в день
• Макс. размер файла: 32 МБ
