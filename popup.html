Stripe Test Autofill

How to use:

  1. Navigate to cursor.com
  2. Look for the floating button in the bottom-right
  3. When on a Stripe checkout page, click the button
  4. Or use keyboard shortcut: Ctrl+Shift+A
⚠️ Test Mode Only

This extension only works with Stripe test mode and will block any attempts to use it in live mode.
