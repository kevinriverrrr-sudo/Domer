?? УСТАНОВКА:
???????????????????????????????????????????????????????????????????????

1. ТРЕБОВАНИЯ:
   - Android 5.0 и выше
   - GTA SA для Android
   - SAMP Mobile
   - Monetloader (последняя версия)

2. УСТАНОВКА MONETLOADER:
   а) Скачайте Monetloader для Android
   б) Установите следуя инструкциям
   в) Убедитесь что Monetloader активирован

3. УСТАНОВКА СКРИПТА:
   а) Скопируйте "PoliceHelper_Monetloader.lua" в:
      /sdcard/Android/data/com.rockstargames.gtasa/moonloader/
   
   б) Или используйте файловый менеджер на телефоне
   
   в) Перезапустите игру

4. ПРОВЕРКА:
   - Зайдите в SAMP
   - В чате появится сообщение о загрузке скрипта
   - Используйте /mhelper для проверки

?? КОМАНДЫ:
???????????????????????????????????????????????????????????????????????

ОСНОВНЫЕ:
/mhelper или /mh         - Открыть/закрыть меню
/mph                     - Быстрое открытие

БЫСТРЫЕ ДЕЙСТВИЯ:
/msu [id] [stars] [reason] - Объявить розыск
   Пример: /msu 5 3 Ограбление
   
/marrest [id]            - Быстрый арест
   (использует данные последнего /msu)
   Пример: /marrest 5

?? ИСПОЛЬЗОВАНИЕ:
???????????????????????????????????????????????????????????????????????

1. БИНДЕР КОМАНД:
   - Откройте меню через /mhelper
   - Перейдите в раздел "Биндер"
   - Выберите нужное действие
   - Заполните параметры если требуется
   - Нажмите на действие для выполнения

2. КОДЫ ПРАВОНАРУШЕНИЙ:
   - Раздел "Коды"
   - Просмотр всех кодов с информацией:
     * Номер кода
     * Название преступления
     * Количество звёзд
     * Размер штрафа
     * Время заключения

3. БЫСТРЫЕ ДЕЙСТВИЯ:
   
   А) Розыск через меню:
      - Откройте раздел "Быстрые"
      - Используйте кнопку "Объявить розыск"
   
   Б) Розыск через команду (РЕКОМЕНДУЕТСЯ):
      /msu 5 3 Ограбление банка
      ?
      ID игрока: 5
      Звёзды: 3
      Причина: Ограбление банка
   
   В) Быстрый арест:
      /msu 5 3 Ограбление  <- Сначала объявите розыск
      /marrest 5           <- Затем арестуйте
      
      Скрипт автоматически:
      - Наденет наручники с /me
      - Объявит розыск
      - Произведёт арест

4. 10-КОДЫ:
   Быстрые кнопки в разделе "Быстрые":
   - 10-4 (Принято)
   - 10-6 (Занят)
   - 10-7 (Вне службы)
   - 10-8 (Свободен)
   - 10-20 (Местоположение)
   - 10-99 (Срочная помощь!)
   - Погоня
   - Подкрепление
